最近折腾了好几天自搭博客，让 AI 帮我跑代码，反复修改什么的，虽然还有些需要完善的地方，但现在终于可以说，博客的迁移工作基本上完成了。

我此前因为嫌折腾自搭博客麻烦，那个时候还没盛行 AI，代码出错了我也不知道该如何从搜索引擎里寻找答案，在很长的搜寻之后，我才敲定了 mataroa、bearblog 这两个平台。mataroa 真的非常非常简洁和纯粹，没有付费会员，没有广告跟踪，还有一个在当时的我非常受益的功能 ———— 博文评论区。每一个博文都自带一个可以匿名的评论区，我当时也是折腾外部评论插件折腾烦了，发现这个功能之后就把 mataroa 当作我当时的博客留言板块了。

![ee701835-e193-485f-9b0e-5657080a2bd8.webp](https://nightola.pages.dev/file/blog_post/1778484162141_ee701835-e193-485f-9b0e-5657080a2bd8.webp)

从这个留言板弄起来到现在，已经收到了十来条留言，别说，一瞬间真的有种大家都是我网络好邻居的感觉，有点像多年前玩 QQ 空间的时候经常给人留言的氛围。想起来我之前那个 QQ 上面还有以前同桌当时给我的留言呢。

![f31269c8-9b11-451a-b3de-bbfdcfc9415a.webp](https://nightola.pages.dev/file/blog_post/1778491587068_f31269c8-9b11-451a-b3de-bbfdcfc9415a.webp)

不过随着长大，我的 QQ 空间就越来越孤寂了，更多都是我一个人自言自语，写术曲推荐或者分析歌曲/自己的垃圾调声了。自搭博客真的是有种在网络上搭自己的房子的感觉，很喜欢。

而 bear blog 呢，在当时我还是感觉挺好用的，也是无广告无跟踪，没有乱七八糟的东西，主题可以自定义，文章和草稿都从没丢过。只是那时，我多年来已经养成了心里有什么话就发动态的习惯，也就是写微博客，所以博客依然没有什么明显的起色。并且我当时的精神状态不好，曾经写过一些非常负面的想法，但后来我都选择不公开了。

![342768b1-dcfc-4ffc-b7be-c0eeb830b1b0.webp](https://nightola.pages.dev/file/blog_post/1778490411182_342768b1-dcfc-4ffc-b7be-c0eeb830b1b0.webp)

当时有些时间是爱看酷安的，我就去酷安写了几个长图文，写我用的软件和备用机，有一篇还上了头条，然后有三百多个赞。后面看到酷安有年度总结的话题，我这才开始在 2024 年写下我十年前其实非常抵触的年度总结。认真的说，写这东西很费人，能花好几天去想到底该怎么写，写什么，但好在效果还可以，于是这一年一度的环节就坚持了两年。

说到年度总结，十来年前我家里人让我写的时候，好像还是因为公司要求必须写的，其实就是工作方面的总结和规划。我家里人因此就尝试让我写学习方面的总结和规划。现在想想，这真没意思啊，生活怎么可能只有学习，人总要有点精神方面的需求吧。可我家里人的精神世界相对比较匮乏，又确实没什么好写的。而且论我在中学时期的生活，也确实没什么可以写成年度总结的时候，更多的时候还是在学校里学习，偶尔画画、玩游戏、听歌就是我的生活了，不过现在除了画画，好像跟以往的生活没什么区别。但能记录下来已经不错啦。

此后我断断续续就尝试让这个博客多更多的功能，比如那个“睡似了么”的界面。通过 AI 的帮忙，我成功在手机上折腾了 MircoAndroid 脚本，可以在我开启宏的时候手动更新睡着还是醒来的状态，也成功把 last.fm 的一个插件弄到的博客里，这样我听什么歌就可能会让别人看到了。当时我蛮自豪的。我就感觉这个功能好好玩。

![compressed_1778491108699.jpg](https://nightola.pages.dev/file/blog_post/1778491147219_compressed_1778491108699.jpg)

然后我还想办法优化了这个界面，ai 帮我写了 python 脚本，我电脑正在打开的窗口也会被上传到数据库，然后显示在这个页面上。再往后就是帮我写主题、写表单插件什么的，这个博客我越发满意了。但我还是因为一些事情暂时停止了折腾博客。

那么就说回来，关于我为什么迁移博客，原因很简单。因为 Bear Blog 毫无征兆地在我不关注博客的时候禁止我免费使用 JavaScript，把这个权限搬到了付费会员里面，并且还新增了一个 180 多美元（折合人民币 1000 多元）的永久会员。我感觉有可能是在针对我吧，毕竟我没有看见其他使用 Bear Blog 的人像我这样给博客加了乱七八糟的功能，以及我之前在关于我的界面里展示了我有的正版歌姬声库，基本上都是几百或者到一千以上的，我猜测人家可能是觉得我有钱吧，一千多块钱为了 javascript 算什么呢，动动手指头的事情呗是不是（）

![e07ea27b-3077-4a35-9247-dbdb360077f1.webp](https://nightola.pages.dev/file/blog_post/1778489925922_e07ea27b-3077-4a35-9247-dbdb360077f1.webp)

不不不，在我的消费观看来，这跟抢钱没区别。按照一个网络梗来说，那就是，他明明可以直接抢我一千块钱的，结果居然还让我在他开发的博客里用 JavaScript 和多开博客，他好善良我哭死（bushi）如同我之前说的那样，假如我有一千块钱，我宁愿买声库、买游戏/软件或者买更好的便于自己创作的设备，我也不可能再给他一分钱。最近就按照 AI 的指引，成功捡起我几年前就尝试过的自搭博客，把原先在那边博客弄好的东西搬到了这边来，并且还突发奇想加了一个随机术力口的功能。如果有人听并且喜欢我推荐的歌就好啦……

而且我才发现，原来 Hugo 有一个相当好用的功能，那就是模板，写好模板之后，我只需要在 `content/指定文件夹` 里面按照格式添加 markdown 文件（其实还是 yaml），我就可以像填表一样简单轻松地添加新的卡片了。现在的留言、提问箱、应用推荐全都模板化了，以后添加就很方便了。如果我真花那一千块钱，然后还是只能在一个文章里看着密密麻麻的代码去添加内容，那可太吃力不讨好了。

![b0a19b23-e267-4eab-abd5-5a7bca2e9614.webp](https://nightola.pages.dev/file/blog_post/1778490482562_b0a19b23-e267-4eab-abd5-5a7bca2e9614.webp)

显然，我在博客上的需求和 bear blog 对博客的定义有所不同，因为 bear blog 本该是用来只写文字和输出内容的，而我想让我的这个网络房子有更多的装饰品和别样的内容。我该离开了。

其实我之前就在 Bear Blog 开过一个月的会员，当时买了域名，然后就是试试域名绑定的效果，当时还是蛮不错的。不过后来我还是懒得给域名续费了，结果不曾想，几年之后我再尝试打开那个域名，想看看有没有人用这个域名，结果映入眼帘的就是不良网站了。唉，可惜这么好的一个词啊。

再说到我怎么开始想写博客，感觉能追溯到当初我折腾笔记软件的时候。我后来就听说了 obsidian 和 Notion 这两个工具，并且也使用过一段时间。obsidian 是每一篇笔记都是 markdown 文件，我用着用着就习惯了 markdown 语法，当然也是在没有 AI 的时候。其实现在 AI 爱用 markdown 标记我还挺诧异的。Notion 的话，本来也想用作我的网络知识库之类的，但最后也没有完全成型，而我在网络上知道它可以弄成一个免费的在线博客的时候，也折腾过，再后来我就尝试了自搭博客，再后面的故事就不用多讲了。

我感觉自己到底是因为比较孤独吧，所以才比较执着于整博客，想在博客分享我的日常、照片、听过的歌、用过的软件，以及我的感受。我还真没听说过有同学整个人博客的。不过我现在跟他们也和切断联系差不多了。旧人旧事就永远留在过去吧。

我去年就给自己整了属于自己的图床，今年则是实现了把博客数据也掌握在了我自己的手上。这么想着，反而我还要稍微感谢一下 Bear Blog 的不合理定价，才让我做出现在的决定并付诸于行动。我也借这个机会去看了看以前的博文，显然不是很多，但都是我当时一个字一个字打出来的。我想是时候更加丰富一下这个博客了。

突然反应过来，其实直接在 GitHub 的公开仓库上写草稿的话，就算删了也是会有痕迹的，还没写好的文章也会被提前看到。但，下次再用私有仓库去写吧。
